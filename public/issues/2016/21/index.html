Multiple URL performance testing support for Azure Web Applications
Performance/load testing of Azure web applications is supported for a while, today we are announcing support for multiple URLs performance testing using Visual Studio web test format. Typically web application includes multiple web pages with headers and query parameters, so currently user has to create multiple test runs in Azure portal for testing each of these pages. We have been getting feedback to support multiple web pages in a single performance test run. With the latest update, you can now simulate a user scenario with multiple URLs and performance test your web application fully.
